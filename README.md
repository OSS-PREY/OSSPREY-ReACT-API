# ReACT Extractor

For hosting ReACTs (actionables/suggestions) for OSSPREY projects

## Installation

Clone the repository and install in editable mode:

```bash
git clone https://github.com/OSS-PREY/OSSPREY-ReACT-API
cd react-extractor
pip install -e .
```
